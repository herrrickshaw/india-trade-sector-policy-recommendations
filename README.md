# India Trade — Sector &amp; Policy Recommendations

The prescriptive companion to [`india-trade-data-analysis`](https://github.com/herrrickshaw/india-trade-data-analysis) — this repo takes that repo's import/export chapter rankings, country-concentration findings, and TRADESTAT-derived trade data and turns them into explicit, checkable claims about **which sectors need PLI-style intervention, which country relationships need active management, and where the current policy response has genuine gaps** — cross-checked against NITI Aayog's own Trade Watch Quarterly framework for what a rigorous version of this kind of analysis should also cover.

Both repos were originally one (`mospi-dataset-analysis`); they were split so the descriptive "what does the data show" analysis and the prescriptive "what should be done about it" recommendations could be published, versioned, and read independently.

**Live site**: [herrrickshaw.github.io/india-trade-sector-policy-recommendations](https://herrrickshaw.github.io/india-trade-sector-policy-recommendations/) — every chart and JSON dataset below served directly via GitHub Pages, no build step.

## Abstract: why import substitution — the forex position, the plan, and the precedent

[`charts/abstract_forex_import_substitution.html`](https://herrrickshaw.github.io/india-trade-sector-policy-recommendations/charts/abstract_forex_import_substitution.html) — open in a browser — the opening frame everything below hangs from, structured as an eight-act narrative following the plan-then-write, narrative-function storytelling model (arXiv:2406.00554): each act carries exactly one narrative function, and every scene is carried by a verified figure rather than rhetoric.

**Act 1 (the stakes)**: India imports $776.0bn and exports $441.7bn — a **−$333.2bn deficit that widened 17.5% in a single year**, with mineral fuels alone 26.2% of the import bill, China a −$112.2bn bilateral gap, and the rupee down 11.9% in twelve months. **Act 2 (the plan)**: attract capital in (FDI) and substitute imports out — the ₹1.91–1.97 lakh crore PLI architecture. **Act 3 (the precedent)**: ethanol blending is the completed, PIB-dated proof that import substitution can work in India, in the exact category (fuels) that remains the biggest import head — blending 1.53% (2013-14) → "20 percent in 2025-26, five years ahead of schedule" (PIB Factsheet 150699), **₹1.97 lakh crore of forex saved** (PRID 2283118), ~316 lakh MT of crude substituted, >₹1.66 lakh crore to farmers — with the government's own admitted downsides kept (no consumer price benefit, marginal mileage drop, gated food-vs-fuel, uncommitted beyond E20) and the honest contrast stated: one commodity with one guaranteed buyer does not prove 14 heterogeneous sectors scale the same way. **Acts 4–8** then let this repo's own findings play the conflict (export half works, import half doesn't, 15% disbursed), the twist (the data itself is defective — registered-office FDI, dead IEM series, anonymous beneficiaries), the turning point (implementation ratio 82% → 109.9%, EC-IEM correlation 0.96, Pharma's clean win), the resolution path (six action items), and the continuation (quarterly refresh + the seven-gate schema).

## Beyond PLI: the 2026 non-PLI incentive wave

[`charts/beyond_pli_incentives.html`](https://herrrickshaw.github.io/india-trade-sector-policy-recommendations/charts/beyond_pli_incentives.html) — open in a browser — the first bulletin produced **from the repo's PIB index** rather than by browsing PIB: a full sweep of the 2026 Cabinet-decision register (`data/pib_index.sqlite`, 9,657 releases) for incentives launched **outside the 14 PLI sectors**. **Headline: ~₹1.9 lakh crore of 2026 commitments deliberately avoid production-linked design** — missions, viability-gap funding, sovereign guarantees, credit backstops and plug-and-play land that de-risk *inputs* where PLI pays for *output* (a revealed preference after ~15% PLI disbursal in five years). Three of the four uncovered import heads this repo flagged received instruments this year: **fuels** (Coal/Lignite Gasification, ₹37,500cr, PRID 2260621 — partial, syngas not crude), **fertilisers** (NIPU-2026, PRID 2284800 — direct), **cotton** (₹5,659cr productivity mission, PRID 2258111 — fibre, not apparel), plus **pulses** (₹11,440cr mission). **Plastics (HS39) and inorganic chemicals (HS28) are now the only large uncovered import heads.** Other finds: BHAVYA's ₹33,660cr for 100 plug-and-play parks aims at exactly the land+clearance gate whose EC flow correlates 0.96 with implementation; the ₹12,980cr Bharat Maritime Insurance Pool quietly hardens the Russia-crude lane against insurance exclusion; the FDI land-border-country amendment (PRID 2237806) defines the Beneficial-Owner rule every China-linked JV must clear. Caveat kept: all of this is gate-1 Cabinet commitment — nothing counts as delivered until it shows at gates 4–7.

Underlying data: [`data/beyond_pli_incentives_2026-07-19.json`](https://herrrickshaw.github.io/india-trade-sector-policy-recommendations/data/beyond_pli_incentives_2026-07-19.json).

### Five years of incentive launches (2021–2026): the migration away from production-linked design

[`charts/pib_five_year_incentive_evolution.html`](https://herrrickshaw.github.io/india-trade-sector-policy-recommendations/charts/pib_five_year_incentive_evolution.html) — open in a browser — extends the sweep to five and a half years: the PIB index was backfilled to **83,734 releases (2020–2026)** by five parallel workers, and five classification agents swept each year's Cabinet register (~178–257 rows/yr) into **93 classified incentive launches** with sectors, instruments, outlays and PRIDs. **Headline: the 2026 "de-risking pivot" is the end of a five-year trend, not a blip — new production-linked schemes fell 11 (2021) → 1 → 1 → 0 (2024) → 2 (2025, both hybrids) → 0 (2026 YTD), and even at the 2021 peak, production-linked outlay was only 11.2% of the year's newly committed money** (RDSS/AMRUT/SBM-U missions carried 6× the rupees with none of the narrative). The "linked" payout base migrated output → design → **employment** (ELI 2025, ₹99,446cr — larger than any single PLI). Other five-year finds: coal gasification escalated 4.4× on the same repo-flagged gap (₹8,500cr 2024 → ₹37,500cr 2026); a critical-minerals chain assembled in one year (mission ₹34,300cr + recycling + rare-earth magnets); maritime became a ₹69,725cr package sector without ever getting a PLI; and 2024 — zero new PLI, all capex — is the pivot year in hindsight.

### The matured cohort — 18 schemes whose returns are already in

[`charts/matured_scheme_cohort.html`](https://herrrickshaw.github.io/india-trade-sector-policy-recommendations/charts/matured_scheme_cohort.html) — open in a browser — the PIB register extended to **122,141 releases (2017–2026)** and three tracer agents followed 18 major schemes launched 2008–2020 launch-to-outcome with PRID-cited trajectory waypoints (the method that built the ethanol case, applied to the whole cohort). **Scoreboard: 8 closed-success / 8 partial / 2 laggards** — and every closed-success ran on a **demand-side guarantee or procurement design**: administered price + offtake (ethanol: 20% blending 5 years early, ₹1.97L cr forex saved), bulk procurement (UJALA: LED prices ₹310→₹38, 36.87cr bulbs), revenue-share auctions (coal: 15 tranches, 100+ MT), VGF route auctions (UDAN: 74→165 operational airports), credit guarantees (MUDRA: ₹40.07L cr disbursed), capex+tariff (M-SIPS→LSEM: mobile production 28×, exports 127×, imports 75% of demand → 0.02%). ## Session findings (as of 2026-07-18)

Every headline insight in this repo, pulled up to one place and ordered to match the sections below. Each linked section still carries its own full narrative, methodology, and **underlying data source** — this index is the "insights" half; the data-source citation stays attached to its own section below, kept separate from this summary.

- **[Are import-dependent sectors getting government attention?](#are-import-dependent-sectors-getting-government-attention)** — The fastest-growing high-value import (fertilisers, +118.9%) had no production-capacity scheme until three days before this analysis. Electrical machinery (HS85) has the strongest matched coverage (Semiconductor Mission 2.0, ₹76,000cr); mineral fuels (26.2% of the entire import bill), plastics, inorganic chemicals, and aircraft & spacecraft have no matching scheme at all.
- **[Where India's forex actually goes — country by country](#where-indias-forex-actually-goes--country-by-country-with-a-decade-of-policy-history)** — China alone is a -$112.2bn bilateral deficit, more than double the next-largest, unmoved by six years of Atmanirbhar Bharat/PLI. Russia is the fastest-moving entry (~2.5% to 35.8-35.9% of crude imports in four years, market-driven, no scheme behind it). UAE and Switzerland's deficits are structural gold/gems re-export flows, not substitution gaps.
- **[Priority trade partners by import-growth sector, and real PLI coverage](#priority-trade-partners-by-import-growth-sector-and-real-pli-coverage)** — China supplies 8 of 12 growth chapters ($102.4bn combined) while PLI schemes cover the money-value of only 2 of the 12 outright. UAE ($56.6bn, 7 sectors) and Russia ($52.6bn, 3 sectors) round out the top tier; seven of twelve growth chapters have no PLI sub-scheme at all.
- **[Priority export destinations by growth sector, and the export side of PLI](#priority-export-destinations-by-growth-sector-and-the-export-side-of-pli)** — The USA is the top-8 destination in 11 of 12 growth export chapters ($63.9bn combined) — a bigger concentration than China's grip on imports. PLI Auto and PLI Bulk Drugs are correctly targeted; PLI Textiles is a genuine mismatch, built for MMF apparel rather than the traditional cotton-based apparel (HS61/62) that actually exports.
- **[Sector &amp; Policy Recommendations](#sector--policy-recommendations)** — Two findings validated against NITI Aayog's Trade Watch that don't appear anywhere else in this repo: a **$20.7bn CBAM export risk** on iron/steel (HS72/73), and an independently-corroborated **structural decline in gems &amp; jewellery exports (-29.9%)**. Also states explicitly what this repo doesn't compute (RCA, Trade Intensity Index) rather than presenting a home-grown ranking as equivalent rigor.
- **[Beyond PLI: state incentives and emergent state hubs](#beyond-pli-state-incentives-and-emergent-state-hubs)** — Central PLI is only half the picture: states layer their own capital subsidies, SGST reimbursement, and land/power concessions on top (e.g. Gujarat tops up ISM semiconductor subsidy by 40%, UP by 50%). Real hubs are emerging with hard evidence behind each: Gujarat/Assam for semiconductors (Micron, Tata Electronics), Odisha/Karnataka for steel, Tamil Nadu/Gujarat/Karnataka/Telangana for auto-EV-battery, Telangana/Gujarat for pharma, Andhra Pradesh/Gujarat/Tamil Nadu/Maharashtra for white-goods components. One state (West Bengal) legislatively revoked every industrial incentive it had in 2025 — a genuine finding, not a data gap.
- **[Top 10 FDI sectors: country strategies, game theory &amp; tariffs](#top-10-fdi-sectors-country-specific-strategies-a-game-theory-frame-and-the-official-tariff-timeline)** — Extends the 5-sector framework to 10 (adds Inorganic Chemicals, Plastics, Aircraft &amp; Spacecraft, Medical Devices) with real target FDI-source countries per sector, sourced and dated. Frames India's May-2026 AC-compressor import quota as a live Generalized Nash Equilibrium tariff-rate-quota game (Passacantando &amp; Raciti, 2025); the official tariff timeline shows the entire 2025 US reciprocal-tariff regime was struck down by the Supreme Court on 20 Feb 2026, and its flat-10% replacement itself expires 24 Jul 2026. Textiles &amp; Apparel gets an honest non-finding: no credible FDI-source country exists for it.
- **[Did the money actually show up?](#did-the-money-actually-show-up-fdi-equity-inflow-vs-pli-scheme-launch-dates)** — Tests real DPIIT/RBI FDI equity inflow against each PLI scheme's launch date, cross-checked against IBEF. Result: genuinely mixed — Electronics and Medical Devices show real gains, Steel/Auto/White Goods show none, Pharma's rise looks more like COVID reshoring than PLI. Bigger problem: DPIIT's state-wise data is built from registered-office address, not factory location — proven directly via a Jio Platforms/Gujarat case that predates the Semiconductor Policy by two years, and several "hub" investments (Tata Electronics Assam, Tata/Reliance battery gigafactories) are domestic capital that can never register as FDI at all.
- **[Master sector scorecard](#master-sector-scorecard)** — One table, 10 sectors: FDI-target countries, actual import/export leaders, policy levers, FDI status, and a traffic light. 2 green (Electronics, Medical Devices), 5 amber, 3 red (Inorganic Chemicals, Plastics, Textiles).

## Where India's forex actually goes — country by country, with a decade of policy history

[`charts/country_trade_deficit_and_policy_history.html`](https://herrrickshaw.github.io/india-trade-sector-policy-recommendations/charts/country_trade_deficit_and_policy_history.html) — open in a browser — ranks India's FY2025-26 bilateral trade balance by partner country (Commerce Ministry TIA Portal), then traces the policy history behind the five biggest deficit relationships back several years — not just the six-month window used above — to check whether sustained attention has actually narrowed each gap. Was also prompted by a look at the Commerce Ministry's **NIRYAT portal** (niryat.gov.in) for visualization ideas; direct access was blocked at the TLS level (the same DGFT-family bot protection this analysis has run into before), so its World Map/Commodity/Table-view design and traffic-light target-tracking pattern are described here from public documentation rather than a first-hand look.

**Headline: China alone is a -$112.2bn bilateral deficit — more than double the next-largest — and six years of Atmanirbhar Bharat/PLI hasn't closed it.** China still supplies an estimated 62% of India's electronic components despite PLI (launched March 2020) and its 2023 expansion to 14 sectors. **Russia is the fastest-moving entry on the list**: with no formal import-substitution scheme behind it, Russian oil went from ~2.5% of India's crude imports before 2022 to 35.8–35.9% in each of the last two fiscal years — a market-driven shift (discounted pricing, settled in UAE dirhams to route around sanctions exposure) that made Russia India's #2 deficit country from a near-zero base in four years. **UAE (-$26.5bn) and Switzerland (-$23.1bn)** are dominated by gold/rough-diamond bullion flows tied to the gems-and-jewellery re-export trade — a deliberately open door, not an import-substitution gap. **Indonesia and Malaysia (palm oil)** show the sharpest contrast between effort and outcome: the National Mission on Edible Oils - Oil Palm has run since August 2021 (₹11,040 crore), yet India remains roughly 57% import-dependent on edible oil, costing an estimated $20.56bn/year in forex.

**Validated (2026-07-18)** against TRADESTAT's own country×commodity data for the four countries anchoring this argument (China, Russia, UAE, USA) — the two independent Commerce Ministry sources agree to within rounding on every country. The chapter-level detail sharpens the picture: China's imports are **58% electrical machinery + mechanical machinery (HS85/HS84)** — industrial inputs feeding India's own manufacturing and export lines, not consumer goods, which is exactly why a currency-settlement change wouldn't touch this deficit. Russia's imports are **86% mineral fuels (HS27)** alone. UAE's imports turn out to be an almost even split between **gems/pearls (HS71) and mineral fuels (HS27)** — not "almost entirely gold" as first read, correcting an earlier pass through this data. And India's own exports to the USA are led by the **same HS85 electrical-machinery chapter** that dominates imports from China (+61.5% growth) — consistent with a components-in, finished-goods-out assembly pattern, not a substitutable consumer import.

## Sector &amp; Policy Recommendations

[`charts/sector_and_policy_recommendations.html`](https://herrrickshaw.github.io/india-trade-sector-policy-recommendations/charts/sector_and_policy_recommendations.html) — open in a browser — the capstone of this repo: turns all four bulletins above into explicit, tiered recommendations — which import-growth chapters most need substitution policy, which export chapters to double down on vs. fix vs. treat as open opportunities, and which country relationships need de-risking vs. diversifying vs. monitoring — cross-checked against NITI Aayog's Trade Watch Quarterly (both the Q1 FY2024-25 edition supplied for this analysis and the 8th edition/Q4 FY2025-26 summary via PIB press release).

**Headline findings not covered anywhere else in this repo:**
- **A genuine forward export risk this repo hadn't flagged before**: the EU's Carbon Border Adjustment Mechanism (CBAM) targets exactly the emissions-intensive sectors — iron/steel, aluminium, cement, fertilizer, hydrogen, electricity — where India already exports strongly. HS72 + HS73 (iron/steel and its articles) combine for **$20.7bn of CBAM-exposed export value**; Specialty Steel PLI scales *production*, but doesn't address the *carbon-cost* risk CBAM creates on the demand side.
- **A structurally declining export sector, independently corroborated**: this repo's own HSN data shows gems &amp; jewellery (HS71) exports down **-29.9%** over 8 years; NITI Aayog's Trade Watch separately flags several labour-intensive export sectors (including pearls/gems) as having lost global market share since 2015. Same finding, two completely different methods — reason to take it seriously rather than treat it as a one-year blip.
- **An explicit methodology gap, stated rather than glossed over**: NITI Aayog computes Revealed Comparative Advantage (RCA) and a Trade Intensity Index (TII) using global commodity-share/GDP-weighted data (UN Comtrade/WTO) this repo hasn't collected. This repo's country/sector rankings are dollar-value and PLI-coverage based — a related but methodologically distinct concentration measure, not a substitute for RCA/TII.
- **The official 13-sector HS taxonomy** (from the Commerce Ministry's own TIA Portal user manual, Annexure 1) is reproduced in full inside the bulletin — confirms this analysis's HS84+HS85 "electronics" grouping and standalone HS71 "gems & jewellery" treatment match DGCI&S's own official sector definitions.

## Top 10 FDI sectors: country-specific strategies, a game-theory frame, and the official tariff timeline

[`charts/top10_sector_country_strategies.html`](https://herrrickshaw.github.io/india-trade-sector-policy-recommendations/charts/top10_sector_country_strategies.html) — open in a browser — extends the 5-sector FDI framework above to 10, adding two ground-floor sectors already flagged Tier-1 in the import-substitution ranking (Inorganic Chemicals, Plastics) plus Aircraft &amp; Spacecraft and Medical Devices, and one deliberate non-finding (Textiles &amp; Apparel). For every sector, this bulletin names specific target FDI-source countries with sourced, dated evidence — not a generic "attract FDI" list — and adds two things the earlier bulletins didn't cover: a game-theoretic frame for how India is now using tariff-rate quotas, and a dated timeline of *official* (non-speculative) 2025-2026 tariff actions.

**Country strategies, sector by sector**: Japan is the single most recurring FDI-source country, appearing as a credible target in 6 of the 10 sectors (Electronics, Specialty Steel, Auto/EV/Battery, White Goods, Inorganic Chemicals, Plastics, Medical Devices) via a consistent mechanism — the India-Japan Industrial Competitiveness Partnership (IJICP) and sector-specific METI/AMED memoranda. Korea anchors Specialty Steel (POSCO-JSW, $7.73bn) and Auto/EV/Battery. The USA is strongest in Pharma (BIOSECURE Act-driven CDMO relocation) and Aircraft &amp; Spacecraft. Two assumptions from earlier research were corrected on closer sourcing: **Germany is exiting, not entering, Specialty Steel** (thyssenkrupp divesting its Nashik business to Japan's JFE, not investing), and **Israel's pharma relationship runs backward** (Sun Pharma, an Indian company, acquired Israel's Taro Pharmaceutical — not the reverse). **Textiles &amp; Apparel gets an honest non-finding**: no credible FDI-source country exists for cotton-based apparel — PLI Textiles structurally excludes it, and the real capital movement is Indian-owned capacity reshoring *from* Bangladesh, not inbound foreign investment.

**A game-theory frame for India's own new instrument**: applying Passacantando &amp; Raciti's Generalized Nash Equilibrium tariff-rate-quota model (*Journal of Dynamics and Games*, Apr 2025, DOI [10.3934/jdg.2025011](https://www.aimsciences.org/article/doi/10.3934/jdg.2025011)) to India's own May-2026 DPIIT Quality Control Order on AC/refrigerator compressors — which caps Chinese import volumes at 30-40% of FY2025 levels with the explicit stated goal of forcing China's GMCC (the dominant global maker) to build Indian factories or lose the market. The same frame flags a real tension in the Specialty Steel strategy above: Japan and Korea are simultaneously the two best FDI targets for steel technology *and* named targets of India's own steel safeguard duty (through 2028) — read through the model, that's not a contradiction, it's the mechanism pushing their capacity into Indian joint ventures rather than arm's-length exports.

**The official tariff timeline, restricted to confirmed government/court actions only**: the entire 2025-era US "reciprocal tariff" regime on India (which ranged 26-50% across the year) was struck down by the **US Supreme Court on 20 Feb 2026** as exceeding IEEPA executive authority. It was replaced with a flat global 10% Section 122 surcharge — itself due to expire **24 Jul 2026**, with a proposed 12.5% Section 301 replacement naming India still pending as of retrieval. Because Vietnam's and Bangladesh's reciprocal rates were struck down in the same ruling, **the widely-assumed "India lost competitiveness to Vietnam/Bangladesh on tariffs" story is currently moot** — all three now face the identical flat 10%. One clear bright spot: the US Section 232 100% pharma tariff explicitly excludes generics and biosimilars, a genuine opening given India's generics-heavy export base. And — sourced directly from NITI Aayog's own Trade Watch, not an external critique — **China+1 diversification has had "limited success so far" for India specifically**, with Vietnam, Thailand, Cambodia and Malaysia capturing more of the relocation away from China.

## Did the money actually show up? FDI equity inflow vs. PLI scheme launch dates

[`charts/fdi_vs_pli_launch_correlation.html`](https://herrrickshaw.github.io/india-trade-sector-policy-recommendations/charts/fdi_vs_pli_launch_correlation.html) — open in a browser — the accountability check behind every PLI figure cited elsewhere in this repo: does REAL inbound FDI equity (DPIIT/RBI-tracked money that actually arrived) rise after a scheme launches, or is "committed PLI investment" a different, mostly self-reported number that doesn't show up as foreign capital at all? Built from 8 parallel research passes, each required to cite primary DPIIT sector/state FDI tables and PIB launch announcements directly, and cross-checked line-by-line against [IBEF's published FDI benchmark page](https://www.ibef.org/economy/foreign-direct-investment).

**Headline: the result is genuinely mixed, not the clean success story PLI messaging implies.** Electronics and Medical Devices show real, sustained FDI-equity gains that move independently of India's national FDI cycle — the strongest evidence of an actual policy effect. Pharma rose too, but the timing (the biggest jump landed in FY2020-21, before either pharma PLI scheme had disbursed anything) points more to COVID-era API-security reshoring than to the scheme itself, and FDI has been *falling* the last two years while PLI's own investment-realization figures keep climbing — a real decoupling. Specialty Steel, Auto/EV/Battery, and White Goods show **no sustained rise at all** — Steel's FDI collapsed ~80% in the two years right after its 2021 launch; Auto's one big year (+327%) fully reverted within twelve months. A three-sector no-PLI control group (Inorganic Chemicals, Plastics, Aircraft & Spacecraft) fell −44.6% over the same window against the six PLI sectors' +30.7% average — but that gap is carried by only two or three sectors, not all six.

**The state-level picture has a bigger problem than a mixed result — the data itself may not be trustworthy for this question.** DPIIT's own metadata document states "Documentation on methodology: Not applicable," and state attribution is built from a company's *registered office* address, not its factory location. This research found direct proof: at least 9 of Gujarat's top-10 FDI deals in FY2020-21 were a single company, Jio Platforms (registered in Ahmedabad, real HQ in Navi Mumbai) — which is why Gujarat topped the national FDI table two years *before* its Semiconductor Policy existed. Separately, several flagship "hub" investments already published in this repo — Tata Electronics' Assam ATMP, Tata's Agratas and Reliance's Jamnagar battery gigafactories, Tata Steel's Kalinganagar expansion — are domestic capital that structurally cannot register as FDI at all (Assam's actual FDI equity inflow in FY2024-25: **$2.88 million**, against a headline ₹27,000 crore project). A flat or negative FDI verdict does not mean the investment didn't happen; it means this particular data series can't see it.

**Against IBEF specifically**: IBEF's #2-ranked FDI sector, "Computer Software/Hardware" ($124.6bn cumulative), turns out to be 99.9%+ software — the "Computer Hardware" sub-component is just 0.10% of that total, confirming it's the wrong proxy for electronics-manufacturing FDI entirely. None of the four states this repo has independently documented as real manufacturing hubs (Odisha, Telangana, Andhra Pradesh, Assam) crack IBEF's national top-5 state list — not because the hubs aren't real, but because their flagship investors are domestic capital. And IBEF's own headline claim — a 69% rise in manufacturing FDI from the 2004-14 decade to 2014-24, credited to "Make in India, PLI" — is exactly the kind of loose comparison this bulletin exists to stress-test: PLI covers at most the final 3-4 years of that 10-year window.

## The case for domestic capital in some states, FDI in others

## Report synthesizer (JSON → PDF/Word/PPT)

[`scripts/report_synthesizer.py`](scripts/report_synthesizer.py) — turns any data JSON in this repo (or a new one you drop in, following the same loose "purpose/sectors/findings" shape) into a standalone **PDF**, **Word (.docx)**, and **PowerPoint (.pptx)** report, so that when the underlying figures get refreshed, anyone using this repo can regenerate a shareable report without hand-building slides. Two modes:

- **`--html <chart.html>`** — renders the paired chart bulletin exactly as designed via headless Chrome's print-to-PDF, for a PDF that matches the published page pixel-for-pixel.
- **JSON-only (no `--html`)** — walks the JSON generically (nested objects become headings and subsections, lists of similarly-shaped objects become tables, lists of strings become bullets) to build the Word and PowerPoint versions, and a plain PDF if no HTML is available. This is deliberately schema-agnostic: it doesn't need per-bulletin templates, so it keeps working as new bulletins with new JSON shapes are added.

```bash
cd scripts
pip install -r requirements.txt      # python-docx, python-pptx, pypdf
python3 report_synthesizer.py ../data/fdi_vs_pli_launch_2026-07-19.json \
  --html ../charts/fdi_vs_pli_launch_correlation.html \
  --formats pdf,docx,pptx \
  --out-dir ../reports/
```

Or batch-run every bulletin in the repo via the manifest:

```bash
python3 report_synthesizer.py --manifest ../data/repo_manifest_2026-07-19.json --formats pdf,docx,pptx --out-dir ../reports/
```

## Methodology notes

- **This repo makes recommendations; it does not make policy.** Every tier/priority label here is this repo's own transparent, stated scoring choice (chapter growth × value × existing-scheme status, or country combined-exposure × sector breadth) — not an official government risk assessment or a validated economic model.
- **RCA and TII are explicitly not computed here** — see the Sector & Policy Recommendations bulletin's "what this analysis doesn't do" section for why, and what would be needed to add them (UN Comtrade / WTO global trade-share data).
- **The NITI Aayog Trade Watch PDF used for structure/methodology is the Q1 FY2024-25 edition** (April–June 2024) — one to two years older than this repo's FY2025-26 claims, so it was used for framework and qualitative corroboration (e.g. China as largest import source, USA as largest export destination — both independently confirmed), not exact-figure validation.
